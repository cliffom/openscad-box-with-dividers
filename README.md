# Box w/ Dividers

OpenSCAD script for creating a box (or more accurately, a tray) with or without dividers.

## Example

### Parameters

```
Width: 100mm
Length: 100mm
Height: 30mm

Wall Thickness: 2mm

Rows: 3
Columns: 3
```
